# arXiv:2509.07931 supplemental material

This folder contains all the required scripts and data files needed to
reproduce the figures presented in arXiv:2509.07931.

The software we used is a branch of the original
[Pyqcm](https://bitbucket.org/dsenechQCM/qcm_wed/src/master/) library also
available [here](https://github.com/antoinedelagrave/qcm_wed) on GitHub.
